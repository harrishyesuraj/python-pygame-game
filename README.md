
# Python Game Using Pygame

## Academic Project (First Year)

This project is a basic 2D game developed using Python and the Pygame library
as part of my first-year academic coursework.

## Features
- Keyboard-based player movement
- Game loop implementation
- Basic collision detection
- Score tracking
- Simple graphics and sound effects

## Technologies Used
- Python
- Pygame

## Learning Outcome
- Understanding of game development basics
- Improved Python programming skills
- Experience with event handling and graphics rendering
